# FarmTech Solutions – Análise de Dados Agrícolas
Projeto desenvolvido com o objetivo de simular um sistema de gerenciamento de plantações e análise de dados agrícolas utilizando Python e R.

## Funcionalidades
O sistema permite:

* Inserir dados de plantações
* Visualizar registros cadastrados
* Atualizar informações de plantação
* Deletar registros
* Calcular manejo de insumos agrícolas
* Exportar dados para CSV
* Consultar dados meteorológicos via API
* Realizar análise estatística com R

## Tecnologias utilizadas
* Python → sistema principal e CRUD
* R → análise de dados e integração com API meteorológica
* CSV → armazenamento dos dados
* API meteorológica → consulta de clima em tempo real

## Como executar o projeto
1. Clone o repositório: git clone https://github.com/AntunyDev/Atividade-Avaliativa-farmTech.git
2. Acesse a pasta do projeto: cd IA_CAP1/python
3. Execute o sistema: python main.py

## Exemplo de saída

- Cidade: Conceição da Aparecida - MG 
- Temperatura: 20°C
-  Velocidade do vento: 3.8 km/h
